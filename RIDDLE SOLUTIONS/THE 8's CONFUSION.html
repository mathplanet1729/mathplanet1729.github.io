SOLUTION - THE 8's CONFUSION

QUESTION (Riddle 1)

How can you make 1,000 by using 8 exactly eight times

ANSWER

The trick in this riddle was to understand that it is not mentioned in the question that we have to use 8 as a single digit only! Thus, we can solve it like this:

888 + 88 + 8 + 8 + 8 = 1,000
